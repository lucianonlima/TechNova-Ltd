# TechNova Sales Analysis

This project analyzes company sales data using Python and Pandas.

## Objectives
- Load and clean sales dataset
- Calculate revenue
- Identify top performing products
- Generate business insights

## Technologies
- Python
- Pandas

## Results

Total Revenue: 7690

Top Products:
1. Monitor
2. Desk
3. Chair
4. Keyboard
5. Mouse

## Business Insight
Monitors represent the highest revenue product, suggesting strong market demand in the electronics category.
